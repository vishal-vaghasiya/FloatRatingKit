# FloatRatingKit

A lightweight and customizable iOS rating view written in Swift.

## Features

- Whole Rating
- Half Rating
- Float Rating
- IBDesignable Support
- UIKit Compatible
- Swift Package Manager Support

## Installation

### Swift Package Manager

```swift
.package(
    url: "https://github.com/vishal-vaghasiya/FloatRatingKit.git",
    from: "1.0.0"
)
